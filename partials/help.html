Navigation - Home screen

Tap on the left top left corner to launch the side bar navigation. When you are in the home screen, tap the top right corner icon to login and logout For non homescreen pages, the arrow icon displayed at the top right corner will take you back to the previous page.

Swipe right to navigate to the next page apps. Swipe left to navigate to the previous page.

Launching the Apps

Three ways to launch the apps

 Home screen app icons
 Sidebar menu
 Dock icon

Report an issue

If you encounter an issue with the app, please use the ‘Report Issue’ option located in the sidebar menu. Please attach screenshots if possible to this message.

Troubleshooting

If the MyMassasoit App fails to function properly, user the Troubleshooting option located in the sidebar menu. (You must select yes when prompted to proceed with an application reset) Do not minimize or close the application during this process.
